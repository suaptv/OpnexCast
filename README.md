# OpnexCast
是涂鸦画板、浏览器、本地相册和视频全能投屏小工具

# 软件特点
1. 边画边投：涂鸦画板，AirPlay实时投到大屏
2. 全屏镜像：深度适配 AirPlay / Apple TV
3. URL投屏: 让你投屏一览无余
4. 浏览器直投：Safari AirPlay 全屏显示
5. 首页卡片: 排序方便用户快捷操作
6. 幻灯片DLNA投屏: 带来更好的体验
7. 记住DLNA服务器: 快速连接，省去重复操作的麻烦

OpnexCast 让媒体共享和播放变得前所未有的简单，提供流畅高效的投屏体验。
