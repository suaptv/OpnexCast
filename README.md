# OpnexCast

**一款把创作与投屏融为一体的全能投屏工具**

集 **涂鸦画板、浏览器、本地相册与视频投屏** 于一身，让内容从屏幕到大屏，始终流畅自然。

<br/>

### ✨ 让媒体共享、展示与播放  
### **通过全屏实时投屏变得前所未有的简单**

</div>

## 🚀 核心亮点

- 🎨 **随画随投**：涂鸦画板创作，AirPlay 实时镜像到大屏  
- 🖥️ **真正的全屏镜像**：深度适配 AirPlay / Apple TV，画面完整呈现  
- 🔗 **URL 连续投屏**：支持 URL 投屏不中断，播放稳定从容  
- 🌐 **浏览器即投**：Safari AirPlay 镜像，全屏显示网页内容  
- ⚡ **高效操作体验**：卡片化首页、DLNA 连播、记住设备快速连接  

---

## 🧩 软件特点

### 1) 🎨 随画随投
在涂鸦画板上自由创作，内容通过 **AirPlay 实时镜像到大屏**，所见即所投。

### 2) 🖥️ 真正的全屏镜像
深度适配 **AirPlay 与 Apple TV**，画面完整呈现，大屏显示更沉浸。

### 3) 🔗 URL 连续投屏
支持 **URL 投屏不中断**，连续播放更从容、更稳定。

### 4) 🌐 浏览器即投
**Safari AirPlay 镜像**，全屏显示网页内容，演示与分享更轻松。

### 5) ⚡ 更高效的操作体验
- 🧱 **首页卡片式布局**：排序清晰，常用功能一触即达  
- 🖼️ **DLNA 幻灯片连续投屏**：连播更流畅，展示更自然  
- 📡 **记住常用 DLNA 设备**：快速连接，省去重复操作
